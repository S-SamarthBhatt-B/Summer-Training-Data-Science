# 🧠 Tech Workers & Mental Health

An end-to-end **Data Analytics & Business Intelligence** project that explores mental health trends among employees in the technology industry using **Python**, **Excel**, and **Microsoft Power BI**.

The project demonstrates the complete data analysis pipeline—from raw data cleaning to interactive dashboard creation—helping organizations gain insights into workplace mental health.

---

## 📌 Project Overview

This project analyzes the **OSMI Mental Health in Tech Survey** dataset to identify patterns related to mental health treatment, workplace environment, company benefits, remote work, and demographic factors.

The workflow includes:

- Data Cleaning using Python
- Data Transformation
- Exporting Clean Data to Excel
- Interactive Dashboard Development in Power BI
- Business Insights & Visualization

# 🧹 Data Cleaning Process

The raw survey dataset contained inconsistent and missing values.

The following preprocessing steps were performed using Python:

- Removed invalid age values
- Filtered unrealistic age range
- Standardized Gender values
- Filled missing values
- Removed unnecessary columns
- Corrected data types
- Exported cleaned dataset to Excel

The Power BI dashboard includes:

- Country-wise Analysis

---

# 📈 Key Insights

- Around **50%** of respondents have sought mental health treatment.
- Employees with a **family history** are more likely to seek treatment.
- **Remote workers** show a higher treatment rate.
- Female respondents seek treatment more frequently than males.
- Nearly half of respondents reported that mental health affects their work.
