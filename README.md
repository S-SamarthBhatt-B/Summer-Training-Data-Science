# 🏏📊 Summer Training — Data Science Projects

A collection of **3 Data Science projects** completed as part of my summer training. Each project covers a different domain and demonstrates core data science skills including data cleaning, exploratory data analysis (EDA), and visualization.

---

## 📁 Projects Overview

### Project 1 — IPL Cricket Data Analysis
**Folder:** `Project-1/`

An end-to-end analysis of IPL (Indian Premier League) cricket match data. The notebook explores team performances, player statistics, match outcomes, and trends across seasons using Python.

**Tools & Libraries:** Python, Pandas, Matplotlib, Seaborn  
**File:** `Project_1_IPL_Cricket_Data_Analysis.ipynb`

---

### Project 2 — Ecommerce Sales Analysis
**Folder:** `Project-2/`

A sales analysis of an ecommerce dataset using Excel. Explores revenue trends, top-selling products, category performance, and customer purchase patterns.

**Tools:** Microsoft Excel  
**File:** `Project-2 Ecommerce Sales Analysis.xlsx`

---

### Project 3 — US Elections Dataset EDA
**Folder:** `Project-3/`

Exploratory Data Analysis on the US Presidential Elections dataset (1976–2020) sourced from Kaggle. The project covers data cleaning, handling missing values, statistical summaries, and visualizations of voting patterns and candidate performance across states and years.

**Tools & Libraries:** Python, Pandas, Matplotlib, Seaborn, OpenDatasets  
**File:** `EDA_Project_1.ipynb`

---

## 🛠️ Tech Stack

- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- Microsoft Excel

---

## 🚀 How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/S-SamarthBhatt-B/Summer-Training-Data-Science.git
   ```
2. Open any `.ipynb` file in [Jupyter Notebook](https://jupyter.org/) or [Google Colab](https://colab.research.google.com/).
3. Run the cells in order.

> For Project 3, you'll need a Kaggle API key to download the dataset via `opendatasets`.
